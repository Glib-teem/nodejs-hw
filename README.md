# NodeJS Homework - Express Server - додаток для роботи з колекцією нотаток.

## Технології
- Node.js
- Express.js
- Helmet (безпека)
- CORS
- Pino-http (логування)
- ESLint
- Prettier

## Встановлення
npm install

## Запуск
npm start

### Development
npm run dev

### Production
npm start

## API Endpoints
- `GET /notes` - Отримати всі нотатки
- `GET /notes/:noteId` - Отримати нотатку за ID
- `GET /test-error` - Тестова помилка
